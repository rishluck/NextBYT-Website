NEXTBYT — IMAGE FOLDER GUIDE
=============================

📁 images/
│
├── 📁 portfolio/
│     project-1.jpg      ← main featured portfolio project (wide card)
│     project-2.jpg      ← second portfolio project
│     project-3.jpg      ← third portfolio project
│
├── 📁 services/
│     web-design.jpg     ← Web Design & Development card
│     shopify.jpg        ← Shopify Design card
│     social-media.jpg   ← Social Media Marketing card
│     brand-design.jpg   ← Logo & Brand Design card
│     ai-marketing.jpg   ← AI Marketing card
│     graphic-design.jpg ← Graphic Design card
│     architecture.jpg   ← Architecture & 3D Design card
│     interior.jpg       ← Interior Design card
│
└── 📁 about/
      team.jpg           ← About section team/studio photo

TIPS:
- Recommended size: 1200x800px for service cards, 1600x900px for portfolio
- Keep file size under 500KB for fast loading (use tinypng.com to compress)
- Use .jpg for photos, .png for logos/graphics with transparency
